安装：著名的5分钟安装办法

  1. 解压缩文件到一个空目录。
  2. 使用文本编辑器（比如 WordPad）打开 wp-config-sample.php 文件，填写您的数据库连接细节。
  3. 将文件保存为 wp-config.php。
  4. 全部上传。
  5. 在浏览器中打开 /wp-admin/install.php 。这将为您的 Blog 创建数据表。如果发生错误，请检查您的 wp-config.php 文件，然后重试。如果再次失败，请前往 支持论坛 寻求帮助，并注意带上您能收集到的信息。
  6. 注意保存给您的密码。
  7. 安装脚本将送您到 登录页面。使用用户名 admin 和安装过程中生成的密码登陆。您可以点击“资料”来更改密码。

升级

在升级之前，请确认已经备份所有您修改过的文件，比如说 index.php。

升级任意版本到 WordPress 2.7：

  1. 备份您修改过的文件，删除旧的 WP 文件
  2. 上传新的文件。
  3. 在浏览器中打开 /wp-admin/upgrade.php。
  4. 想要下一步？没了！

从其他系统升级

WordPress 可以从很多别的系统导入数据。首先，您需要按照上面的办法安装 WordPress 。

XML-RPC 和 Atom 接口

您可以通过类似于 Windows Live Writer, Ecto, Bloggar, Radio Userland (这意味着您可以使用 Radio 的 email-to-blog 功能), NewzCrawler, 和其他支持支持 Blogging API 的工具发表文章到您的 Blog！:) 您可以 在 CodeX 上的 XML-RPC支持 来获取更多信息。

通过 Email 发表文章

您可以通过 email 客户端发表文章！要使用此功能，可以通过“撰写”选项页面，填写您的秘密 POP3 帐号。然后您需要设置 wp-mail.php 来定期执行检查这个信箱。您可以通过计划任务来实现；如果您的主机不支持，也可以尝试使用网站监视服务，让他们来检查您的 wp-mail.php 网址。

发表文章还是很容易地：任何发送到指定地址的 email 都会被发表，邮件的标题用作文章的题目。该邮件地址务必保密。如果发表成功，脚本程序会 删除 这些 emails。
